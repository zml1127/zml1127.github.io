# 物品管理平台

## 项目概述

一个基于浏览器的物品管理平台，用于帮助用户管理柜子和物品。系统支持用户认证，采用响应式设计，适配手机端和PC端。

## 功能需求

### 1. 用户认证系统
- **登录功能**
  - 用户名和密码验证
  - 登录状态持久化（Session）
  - 默认管理员账号：admin / admin123

- **登出功能**
  - 清除登录状态
  - 返回登录页面

- **访问控制**
  - 单角色（管理员）访问控制
  - 未登录用户访问管理页面时跳转登录
  - 不包含权限细分

### 2. 仪表盘
- 展示柜子总数统计
- 展示物品总数统计
- 各柜子物品数量分布
- 最近添加的物品列表

### 3. 柜子管理
- 创建新柜子（支持设置柜子名称）
- 删除已有柜子
- 查看所有柜子列表
- 每个柜子显示其内部的物品数量

### 4. 物品管理
- 创建新物品（支持设置物品名称）
- 为物品选择放置的柜子
- 删除物品
- 物品显示其所属柜子的位置信息

### 5. 数据展示
- 仪表盘区域展示统计数据
- 左侧区域展示所有柜子列表
- 右侧区域展示所有物品及其柜子归属
- 清晰标识物品放置位置

## 技术实现

### 架构
- 纯静态页面（HTML + CSS + JavaScript）
- 单页面应用（SPA）架构
- 使用LocalStorage进行数据持久化

### 用户认证
- 前端模拟认证（基于LocalStorage）
- Session管理机制
- 管理员角色标识

### 响应式设计
- 移动端优先设计
- 媒体查询适配不同屏幕
- 移动端汉堡菜单
- 桌面端侧边栏导航

### 数据存储
- 柜子数据：cabinets数组
- 物品数据：items数组
- 用户数据：users数组（含admin用户）
- Session数据：currentUser

## 用户交互流程

### 登录流程
1. 用户访问系统 → 显示登录页面
2. 输入用户名密码 → 验证身份
3. 验证成功 → 进入主页面
4. 验证失败 → 显示错误提示

### 主页面交互
- 顶部导航栏（仪表盘、柜子管理、物品管理、用户信息）
- 仪表盘展示统计数据卡片
- 表单输入创建柜子和物品
- 点击删除按钮移除项目
- 下拉选择框为物品分配柜子

### 登出流程
1. 点击登出按钮
2. 清除Session
3. 跳转登录页面

## 界面设计

### 登录页
- 居中登录卡片
- 用户名、密码输入框
- 登录按钮
- 平台标题和Logo

### 主页面
- 顶部导航栏（白色背景，带阴影）
- 仪表盘区域（统计卡片网格布局）
- 主内容区（左侧柜子、右侧物品管理面板）
- 响应式三栏布局（移动端单栏）

### 配色方案
- 主色：#667eea（蓝紫色渐变）
- 辅助色：#764ba2
- 背景色：#f8f9fa
- 文字色：#333333
- 强调色：#ff4757（删除按钮）

## 响应式断点
- 移动端：< 768px（单栏布局，汉堡菜单）
- 平板端：768px - 1024px（两栏布局）
- 桌面端：> 1024px（三栏布局，完整导航）
